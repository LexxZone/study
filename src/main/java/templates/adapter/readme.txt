 Адаптер - шаблон, который помогает сопоставить реализацию одного сервиса с интерфейсом другого.
 Допустим есть какой-то интерфейс сервиса iService.
 Также существует уже готовая бизнес логика (ServiceLogic) с аналогичным функционалом, но с собственным уже готовым
 кодом.
 Создаем новый класс - адаптер AdapterBusinessToServiceLogic. Он соединяет реализацию бизнес-логики с логикой сервиса.
